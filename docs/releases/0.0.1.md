# Mac Bastion 0.0.1

Initial preview release.

## Downloads

- `MacBastionMenu-0.0.1-macos-arm64.zip`: macOS menu bar app.
- `mbastion-0.0.1-macos-arm64.tar.gz`: CLI executable and runtime library.
- `SHA256SUMS.txt`: artifact checksums.

## Highlights

- Menu bar control for SSH bastion local forwards.
- Shared YAML config for the menu app and CLI.
- Single-file config and split profile includes.
- YAML import/export.
- Validation for duplicate profile names, duplicate local ports, invalid ports, missing required fields, missing identity files, and live local port usage.
- CLI commands for init, list, validate, render-ssh, start, stop, restart, status, logs, import, and export.

## Notes

The first release is built for macOS arm64. SSH authentication is delegated to the system SSH client and `ssh-agent`.

The app is ad-hoc signed, not Apple Developer ID notarized yet. On first launch macOS may require right-clicking the app and choosing Open. If Gatekeeper still blocks a downloaded zip, run:

```sh
xattr -dr com.apple.quarantine /Applications/MacBastionMenu.app
```
